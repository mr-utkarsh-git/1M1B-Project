# GreenVerse AI-IoT Campus Sustainability Platform

A premium, modern, responsive, and visually stunning website for **GreenVerse** - an AI-powered Smart Campus Sustainability Platform.

Built with a $100M unicorn startup aesthetic, utilizing Tailwind CSS, interactive canvas particles, scrolling sways, pulsing hotspot nodes of India, custom cursors, tab sliders, and dynamic dashboards.

---

## 🚀 Live GitHub Pages Deployment

To host this website live on the web using **GitHub Pages** for free:

1. **Create a GitHub Repository**:
   - Go to [github.com](https://github.com) and create a new repository (e.g. `greenverse-sustainability`).
   
2. **Push the Files**:
   - Upload the following files to the main/master branch:
     - `index.html`
     - `style.css`
     - `script.js`
     - `README.md`

3. **Enable GitHub Pages**:
   - Go to your repository **Settings** tab.
   - Scroll down to the **Pages** menu on the left sidebar.
   - Under **Build and deployment**, select **Deploy from a branch**.
   - Under **Branch**, select `main` (or `master`) and `/root`, then click **Save**.

4. **Visit Your Live Link**:
   - Within 1-2 minutes, GitHub will generate a live deployment link (e.g., `https://your-username.github.io/greenverse-sustainability/`).

## 💻 Local Preview

You can run this project locally on your machine with zero setup:
- Simply double-click **`index.html`** or drag it into any modern web browser.
- Alternatively, run a local server:
  - If you have Python: `python -m http.server 8000`
  - If you have Node: `npx serve .`
